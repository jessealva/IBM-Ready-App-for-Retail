Class: colorPicker

ReadyAppSummit. colorPicker

new colorPicker()

A directive that defines a color picker, which displays a variety of colors as selectable buttons. This directive is decorated with two attributes: colorOptions is an array of objects with color names and hex values. colorPicked is an optional external function that will be triggered whenever a color is clicked.
