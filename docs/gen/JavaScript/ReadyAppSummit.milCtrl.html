Class: milCtrl

ReadyAppSummit. milCtrl

new milCtrl()

The basis for every controller in a ReadyApp. It is recommened that every controller in a ReadyApp extends this to recieve useful functionality that's common between ReadyApps.
Properties:
Name Type Description
agent string The browser rendering the app.
canProcessRoutes boolean True if this controller is allowed to process route changes, false if the controller should ignore actions that may cause a route change. This prevents the hybrid view from suddenly changing route unexpectedly while the client is processing native code.

(static) getData(key, expected) → {any}

Used to access the injectedJSON that was supplied via the injectData function. If the data doesn't exist or doesn't exist in the correct type, this function will construct a new object of type expected for you and return it immediately, outputing the appropriate warnings to the console.
Parameters:
Name Type Description
key string The key for the data you want from injectedJSON.
expected constructor The constructor function for the type of value you want; this is used to cross-reference that the value being returned matches the value you're exepcting. So, for example, if you wanted a key that should be of type Array, you would use the constructor Array for this argument.
