Class: textExpander

ReadyAppSummit. textExpander

new textExpander()

A directive that creates a text box that smoothly reveals and hides itself when the title area is clicked. This directive is decorated with four attributes: title specifies the text to be displayed in the title area. text specifies the text that will be shown or hidden. buttonImage is an optional attribute for specifying an image that will exist to the right of the title. buttonRotation is an optional attribute to specify the number of degrees the buttonImage will rotate when the text is shown; negative values indicate counterclockwise rotation.
