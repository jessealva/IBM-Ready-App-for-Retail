Class: starRatingViewer

ReadyAppSummit. starRatingViewer

new starRatingViewer()

A directive that defines a collection of five images that can be displayed in 2 states: full or empty. Typically, this directive should be used to give items "star ratings" on a 5-star scale, with 5 stars being the best and 0 stars being the worst. This directive is decorated with three attributes: rating, the number of items in the collection that you want filled in, fullPath, a path to the image that depicts the "full" state, and emptyPath, a path to the image that depicts the "empty" state. You may change the dimensions of the images by applying CSS rules to img tags that are nested within this starRatingViewer.
