Class: dropDown

ReadyAppSummit. dropDown

new dropDown()

A directive that automates the creation of a stylized drop down menu, functionally equivalent to an HTML so that its data can be acquired by an HTML form. dropData is an object consisting of a name, which is the header text, and values, which is an array of values that will populate the drop-down menu. buttonImage is an optional image that displays on the right side of the menu.
