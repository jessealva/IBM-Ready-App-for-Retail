(static) imagePreloader :number

A provider that allows for placeholder images to be displayed while a full image loads, so the entire view is not waiting on an image download before it can display. This provider is created by a factory known as the imagePreloader.
Type:
  • number
Properties:
Name Type Description
src string The source for the image to be loaded.
state enum The current loading state of the image.

Example
var loader = new imagePreloader();
 loader.src = "image-src.png";
