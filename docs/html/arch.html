Summit Architecture

Summit's architecture is made up of 5 main components:

  • Native Applications (iOS and Andriod)
  • Hybrid Application
  • Estimote Beacons
  • MobileFirst Platform v6.3
  • Cloudant Database


The hybrid application consists of a mixture of native artifacts and AngularJS/HTML/SaSS. The MobileFirst Platform is used as an interface to the Systems of Record and System of Engagements. Cloudant is used to simulate the System of Record and System of Engagement that you would need to interface your mobile application with.

ReadyApp Summit leverages Estimote Beacon technology to create a location aware application, dynamically change the app behavior based on the users location. Additionally, Summit uses Xtify to push notifications to the user when entering a certain zone.

Hybrid Application

The hybrid application is a mixture of native features along with AngularJS/HTML/SaSS code in order to give a blend of native look and feel along with some of the reusability of full bybrid applications.

Using native controllers and views along with native transitions, the native experience can be preserved and the application can interchangeably use native code when performance is a necessity and hybrid when reusability of complex code or algorithms is desired.

MobileFirst Platform

The MobileFirst Platform is used to provide a well encapsulated interface between the mobile clients and backend systems of record and systems of engagement. It gives you the ability to concentrate on providing the best user experience while not sacrificing on security or integration.

You can spend more time perfecting your user experience while being able to access your critical business data exactly the same way from all of your client facing applications.

Cloudant Database

A Cloudant database is used to simulate a system of record and the system of engagement. It contains data used for both authentication and user interactions. As this is only a simulation it does not follow the normal standards of systems of engagement or systems of record and really only acts as a data store.
